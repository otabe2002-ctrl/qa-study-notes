Тестовый сценарий: [«Как поступить в «Школу 21»](https://21-school.ru/admission).

**ТС-01. Оформление заявки на поступление**

**Предусловия:**

- Пользователь имеет Сбер ID
- Пользователь не зарегистрирован на 21-school.ru

**Сценарий:**

1.  Вход через Сбер ID или создание нового
2.  Регистрация аккаунта на сайте

**Ожидаемый результат:**

- Аккаунт создан успешно
- Пользователь авторизован и может переходить к следующему этапу

**ТС-02. Прохождение вступительного этапа:**

**Предусловия:**

- Пользователь зарегистрирован на сайте 21-school.ru

**Сценарий:**

1.  Пользователь открывает страницу вступительного испытания
2.  Проходит игру до конца (все уровни / задачи)

**Ожидаемый результат:**

- Пользователь получает возможность перейти к просмотру видео

**ТС-03. Просмотр видео**

**Предусловия:**

- Пользователь прошел полностью игру

**Сценарий:**

1.  Пользователь переходит к вкладке с видео
2.  Просматривает видео о школе полностью

**Ожидаемый результат:**

- Пользователь получает возможность перейти к заполнению анкеты

**ТС-04. Заполнение анкеты**

**Предусловия:**

- Пользователь прошел игру и просмотрел видео полностью

**Сценарий:**

1.  Пользователь открывает форму анкеты

2. Заполняет все обязательные поля (имя, контактные данные, опыт, мотивация и т.д.)

3. Подтверждает согласие с правилами и политикой конфиденциальности

4. Отправляет анкету

**Ожидаемый результат:**

- Анкета успешно отправлена
- Данные анкеты сохранены в системе

**Постусловия**

- Пользователь может ожидать результатов
- Система фиксирует факт подачи анкеты
