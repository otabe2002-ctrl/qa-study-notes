## **Матрица тестового покрытия по категориям важности**

### **КРИТИЧЕСКИЕ проверки (Tier 1)**

Эти проверки являются основой для функционирования сайта. Их отказ делает сайт полностью неработоспособным.

| Функциональность                                   | Номера проверок | Обоснование                                                                                                                                                                                                                  |
| ------------------------------------------------------------------ | ----------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Доступность и загрузка**               | 1, 23                         | Без доступа к сайту пользователи не получат информацию. Медленная загрузка критична для UX.                                                                  |
| **Основная навигация**                      | 9, 10, 11, 12, 26, 27, 28     | Логотип, ссылки и меню — основа навигации по контенту.                                                                                                                                      |
| **Корректность контента**                | 16, 17, 18                    | Актуальность и правильность информации                                                                                                                                                              |
| **Базовая визуализация**                  | 4, 5, 6, 7                    | Сломанная вёрстка, неправильное выравнивание делают контент нечитаемым.                                                                                                  |
| **Кодировка и параметры**                 | 2, 3                          | Корректная кодировка UTF-8 и язык страницы — фундамент контента.                                                                                                                      |
| **Мультимедиа и интерактивность** | 13, 14, 15                    | Кнопки, всплывающие окна, видео — без них пользователь не может взаимодействовать с контентом. Критично для функциональности. |

**Итого критических: 21**

Обеспечивают хороший UX и охват различных сценариев. Недостаток снижает качество, но не делает сайт неработоспособным.

| Функциональность                                | Номера проверок | Обоснование                                                                                                                                                              |
| --------------------------------------------------------------- | ----------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Кроссбраузерность**                    | 19                            | Chrome, Firefox, Safari, Edge — разные браузеры разных пользователей. Работа во всех обеспечивает доступность. |
| **Адаптивность устройств**           | 20, 21, 22                    | Многие заходят со смартфонов и планшетов. Адаптивность обязательна для мобильной аудитории.          |
| **Производительность**                  | 24, 36, 37, 40                | Кэширование, отзывчивость, скорость навигации и время ответа сервера улучшают UX.                                |
| **Дополнительная навигация**       | 29, 30, 31                    | Кнопки навигации, футер, мобильное меню улучшают пользовательский опыт.                                                |
| **Совместимость ОС и устройств** | 34, 35                        | Работа на разных ОС и устройствах обеспечивает широкий охват аудитории.                                                |
| **Элементы интерфейса**                 | 8                             | Выделение при наведении улучшает интерактивность.                                                                                       |
