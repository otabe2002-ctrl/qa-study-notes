| Функциональность                                      | Назначение                                                                                                                                                           | Для каких практических задач тестировщик использует                                                                                                                                                                         |
| --------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Фильтрация запросов**                       | Отображение только нужных HTTP/HTTPS-запросов по домену, типу, статусу, методу и т.д.                               | Поиск конкретного запроса среди сотен; анализ только API-запросов; проверка запросов к определённому сервису; изоляция ошибок 4xx/5xx                             |
| **Работа с логами**                                | Просмотр, сохранение, экспорт и анализ истории сетевых запросов и ответов                                        | Анализ багов; сбор доказательств для отчёта; сравнение корректного и некорректного поведения; повторный анализ без воспроизведения сценария      |
| **Работа с зашифрованным трафиком** | Декодирование HTTPS-трафика через установку сертификата                                                                           | Проверка содержимого защищённых запросов; анализ токенов, заголовков, payload; проверка корректности авторизации; поиск утечек данных                         |
| **Map Remote**                                                  | Подмена ответа сервера на локальный или другой удалённый ресурс                                                         | Тестирование фронта без бэкенда; проверка UI с изменённым ответом; эмуляция ошибок сервера; тестирование фичи до её релиза на сервере                       |
| **Breakpoint**                                                  | Остановка запроса или ответа до отправки/получения с возможностью редактирования                        | Проверка валидации на сервере; изменение параметров запроса; проверка обработки нестандартных значений; тестирование граничных случаев              |
| **Rewrite**                                                     | Автоматическое изменение заголовков, параметров, тела запроса или ответа по заданным правилам | Массовая подмена токенов; тестирование разных ролей пользователя; проверка обработки изменённых данных; автоматизация негативных сценариев      |
| **Throttling**                                                  | Искусственное замедление интернет-соединения                                                                                          | Проверка поведения приложения при плохом интернете; тестирование таймаутов; анализ загрузки контента при 3G; выявление проблем с retry-механизмами |
