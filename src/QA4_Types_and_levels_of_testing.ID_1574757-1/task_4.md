# Чек-лист Black-box тестирования приложения "Найди и иди"

## SC21-01. Поле ввода названия магазина

☐ Поле принимает буквы (a-z, A-Z, а-я)

☐ Поле принимает цифры (0-9)

☐ Поле принимает спецсимволы (!@#$%)

☐ Поле принимает пробелы

☐ Плейсхолдер отображается в пустом поле

☐ Плейсхолдер исчезает при вводе первого символа

☐ Плейсхолдер возвращается после очистки поля

☐ История поиска сохраняется и доступна при клике на поле

☐ При вводе 1 символа кнопка "Поиск" остаётся неактивной

☐ При вводе 2+ символов кнопка "Поиск" становится активной

☐ Поле принимает очень длинные строки (100+ символов)

☐ Поле принимает только пробелы (требует уточнения ТЗ)

---

## SC21-02. Поле выбора типа магазина

☐ Плейсхолдер "Выберите тип магазина" отображается по умолчанию

☐ Выпадающее меню открывается при клике на поле

☐ В меню отображаются варианты: "Продуктовые магазины"

☐ В меню отображаются варианты: "Магазины стройматериалов"

☐ В меню отображаются варианты: "Аптеки"

☐ Выбранное значение отображается в поле после выбора

☐ Плейсхолдер исчезает после выбора значения

☐ Возможно изменить выбранное значение на другое

☐ Меню закрывается после выбора

---

## SC21-03. Кнопка "Поиск"

☐ Кнопка неактивна при незаполненных обоих полях

☐ Кнопка становится активной при заполнении поля названия (2+ символа)

☐ Кнопка становится активной при выборе типа магазина

☐ Кнопка остаётся активной при заполнении обоих полей

☐ Поиск выполняется при клике на активную кнопку

☐ Результаты отображаются флажками на карте

☐ При поиске без результатов флажков нет на карте

☐ Карта скроллится (вверх, вниз, влево, вправо)

☐ Карта зуммируется при pinch-жесте (приближение)

☐ Карта зуммируется при pinch-жесте (отдаление)

☐ Кнопка неактивна при вводе только 1 символа в название

---

## SC21-04. Кнопка "Построить маршрут"

☐ Кнопка активна при выборе флага найденного магазина

☐ Кнопка активна при выборе только типа магазина (без названия)

☐ Иконка кнопки — изогнутая стрелка

☐ Текст кнопки — "Построить маршрут"

☐ Цвет фона кнопки по умолчанию — белый (#fff)

☐ При нажатии кнопки цвет фона меняется на серый (#586d73)

☐ При нажатии текст кнопки меняется на "Отменить маршрут"

☐ Маршрут отображается на карте после нажатия

☐ Маршрут отменяется при нажатии кнопки "Отменить маршрут"

☐ После отмены кнопка вновь становится "Построить маршрут" с белым фоном

☐ Кнопка неактивна при отсутствии выбранного магазина/типа

---

## SC21-05. Область карты

☐ Карта занимает ровно половину экрана (50% снизу)

☐ Размер карты пропорционален на экранах разных диагоналей

☐ Маршрут рассчитывается и отображается корректно

☐ Маршрут отменяется только кнопкой "Отмена", не другими способами

☐ На маленьких экранах карта занимает 50% нижней части экрана

☐ На больших экранах карта занимает 50% нижней части экран

---

## SC21-06. Общие требования

☐ Незаконченный маршрут восстанавливается при повторном открытии приложения

☐ История поиска восстанавливается при повторном открытии приложения

☐ Геопозиция работает при нормальном уровне батареи

☐ Геопозиция отключается при критическом уровне батареи

☐ Приложение не падает при быстром открытии/закрытии

☐ Приложение не падает при смене ориентации экрана

☐ Интервал восстановления маршрута определён (требует уточнения ТЗ)

☐ Пороговое значение низкого заряда определено (требует уточнения ТЗ)
