# Чек-лист Grey-box тестирования приложения "Найди и иди"

**Предусловие: есть доступ к базам данных в которых хранится информация о магазинах.**

## SC21-01. Поле ввода названия магазина

☐ Поиск возвращает магазины, существующие в БД

☐ Поиск не возвращает магазины, отсутствующие в БД

☐ Поиск корректно обрабатывает различные варианты ввода названия

☐ Поиск возвращает все магазины с одинаковым названием, существующие в БД

☐ Поиск не возвращает архивные / неактивные магазины (если есть статус)

---

## SC21-02. Поле выбора типа магазина

☐ Список типов соответствует данным в БД

☐ Выбор типа возвращает только магазины этого типа

☐ Магазин с несколькими типами корректно отображается в результатах

---

## SC21-03. Кнопка "Поиск"

☐ При поиске отправляется запрос в БД

☐ При некорректном поиске (см. Black-blox) )не отправляется запрос в БД

---

## SC21-04. Кнопка "Построить маршрут"

---

☐ Маршрут строится только для магазинов с валидными координатами в БД

☐ Магазины без координат не позволяют построить маршрут

☐ Некорректные координаты не вызывают падение приложения

## SC21-05. Область карты

☐ Все флажки на карте соответствуют записям в БД

☐ Координаты флажков совпадают с координатами в БД

☐ Магазины с одинаковыми координатами корректно отображаются

☐ Магазины, находящиеся в архиве БД, не отображаются на карте

---

## SC21-06. Общие требования

☐ История поиска соответствует  реальным данным из БД

☐ Удаленный из БД магазин не появляется в истории

☐ Приложение корректно работает при большом количестве записей

---
