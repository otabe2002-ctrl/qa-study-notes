Это репозиторий конспектов и практических материалов по тестированию ПО
- `Foundation.md` — основы QA, методологии, принципы
- `Test_artifacts.md` — тест-кейсы, чеклисты, тест-план, тестовые данные
- `Types_and_levels_of_testing.md` — категории, уровни, анализ требований
- `Defects.md` — дефекты, структура, lifecycle, анти-шаблоны
- `Web_testing.md` — архитектура, отказоустойчивость, кросс-браузерность, HTTPS, куки
- `Backend_testing.md` — REST API, JSON, схемы, сравнение API
- `Mobile_testing.md` — мобильные платформы, дистрибуция, сценарии, утечки памяти
- `Basics_of_Web_Security.md` — матрица рисков, криптография, OWASP, security-тесты
